DebugObject - Properties

The DebugObject has 4 properties (all of which apply before the Start of Frame event):

Enable at start
Enables debug output at start (crashes are handled anyway, but won't be logged).
Show message if path invalid
This will cause a message box to appear if output was enabled, and a crash has been caught, before the log file path has been opened. After the message box is closed, the application will be closed.

Initial path
This is the default log path. It will be opened at the start of the application (and as normal, print an open-log message).

Console enabled
Enable the console at the start of the application (will show all log messages and receive input commands).
